# HA-AWCC-Data-Analyzer
This is a Windows application to analyze coincidence counting data from the HA-AWCC

To analyze HA-AWCC data, copy entire "ha_awcc_app" directory into your Windows machine's C: directory. To run the full analysis, launch the application and click "Analyze Data Folder." Select the folder of data you wish to analyze. Note that the data folder must have a particular structure and files require specific names. The data folder must contain folders called "inner", "outer", and "sum". These folders must contain .RTS or .VER files from INCC.

System parameters can be read in using saved files (examples attached).
